# BitSend Contact Information

Last updated: March 20, 2026

## 1. General Support
For general app support, bug reports, and feature requests:
- GitHub Issues: https://github.com/Adarshagupta/BitSend/issues

Recommended issue template details:
- App version and platform (Android/iOS)
- Chain/network used
- Steps to reproduce
- Error message (if any)
- Screenshots or logs that do not expose secrets

## 2. Privacy Requests
For privacy and data-rights requests, open an issue and include:
- Request type (access, correction, deletion, objection)
- Wallet address(es) or context needed to locate related records
- Jurisdiction (country/state) for rights handling

Suggested title:
- "Privacy Request - BitSend"

## 3. Security Reporting
For responsible security disclosure, include:
- Affected component and version
- Reproduction steps or proof of concept
- Severity and potential impact
- Mitigation suggestions (if available)

Suggested title:
- "Security Report - BitSend"

Do not post seed phrases, private keys, or sensitive credentials in public reports.

## 4. Policy and Legal Questions
For questions about Terms and Privacy:
- Reference documents:
  - docs/legal/terms-and-conditions.md
  - docs/legal/privacy-policy.md
- Submit a question through GitHub Issues with title:
  - "Legal/Policy Inquiry - BitSend"

## 5. Expected Response Window
Response times depend on maintainer availability. For critical security issues, include "Critical" in the title so triage can be prioritized.
